## My personal user config for AstroVim

## 🛠️ Installation

#### Install AstroNvim

```shell
git clone https://github.com/AstroNvim/AstroNvim ~/.config/nvim
```

#### Install this configuration

```shell
git clone https://github.com/zdk/astronvim_config.git ~/.config/nvim/lua/user
```

#### Start Neovim

```shell
nvim
```
